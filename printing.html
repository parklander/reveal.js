## Overall process - find or create a model - slice the model and select filament - run gcode on the 3d printer

Where to Find Models

  1. Prusa Prints
  2. Thingiverse
  3. YouMagine
  4. Thingiverse
  5. MyMiniFactory
## 3d model file types - STL -- Most common -- format is limited to storing polygon information. -- no scale information (slicer assumes base unit is mm) -- no assurance of free from manifold errors - 3MF
